Ingrese los datos para agregar un nuevo Bioma.

  • mi_nuevo_minecraft_id_bioma

  • Este es mi es mi nuevo bioma...

    Agregar Bioma
    ¿Ayuda? Requisitos a pensar. Minecraft id bioma:
    • Solo letras minúsculas, números y remplazando los espacios con guiones bajos ( _ )
    • No debe tener espacios vacíos
    • Máximo 50 caracteres
    Nombre del Bioma:
    • Puede contener letras mayúsculas y minúsculas
    • No debe contener números ni caracteres especiales
    • No debe tener espacios vacíos
    • Máximo 50 caracteres
    Descripción del Bioma:
    • Solo letras y espacios
    • No debe contener caracteres especiales
    • Máximo 300 caracteres
